x402 Reputation Oracle API

Machine-readable, EIP-712 signed developer reputation sold per query in USDC via the x402 HTTP payment protocol.

Endpoints (paid)

RoutePrice (default)Returns
GET /v1/reputation/{addr}$0.01 USDCSigned summary report
GET /v1/reputation/{addr}/full$0.05 USDCFull report + attestation evidence
GET /v1/leaderboard$0.02 USDCSigned leaderboard snapshot
GET /v1/docsfreeAPI metadata JSON

Payment flow

  1. Request without X-PAYMENT → server returns 402 with requirements
  2. Sign EIP-3009 USDC authorization on Base Sepolia
  3. Retry with X-PAYMENT header (base64 JSON payload)
  4. Facilitator verifies & settles → signed report in response

Go client (x402-go)

package main

import (
  "fmt"
  "github.com/mark3labs/x402-go/signers/evm"
  x402http "github.com/mark3labs/x402-go/http"
)

func main() {
  signer, _ := evm.NewSigner(
    evm.WithPrivateKey("0xYOUR_KEY"),
    evm.WithNetwork("base-sepolia"),
    evm.WithToken("0x036CbD53842c5426634e7929541eC2318f3dCF7e", "USDC", 6),
  )
  client, _ := x402http.NewClient(x402http.WithSigner(signer))
  resp, err := client.Get("https://YOUR_HOST/v1/reputation/0xabc...")
  if err != nil { panic(err) }
  defer resp.Body.Close()
  fmt.Println("status", resp.Status)
}
